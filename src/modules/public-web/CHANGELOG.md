# Changelog - Public Web Module

All notable changes to the public-web module will be documented in this file.

## [1.0.0] - 2026-01-12

### Added
- Initial landing page with hero, pricing, and footer sections
- Header component with sticky glassmorphism effect
- Animated teal waves background (CSS radial gradients)
- Trust signals with dynamic stats
- Pricing section with 3 plans (ZDARMA, HRA, AI)
- Footer with 4-column layout
- AuthModal integration for login/register
- Error boundary isolation
- Mobile-first responsive design
- Accessibility features (focus states, reduced-motion support)

### Design Specifications
- Colors: Teal primary (#2CBEC6), Gold accent (#D6A23A)
- Typography: Inter font, 48px hero headline
- Spacing: 4px base unit system
- Breakpoints: 390px, 768px, 1024px, 1280px
- Logo: Off-white variant without slogan
