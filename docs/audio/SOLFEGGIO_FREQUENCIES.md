## Production Notes for Audio Files

### Pure Sine Wave Requirements

**Why pure sine wave?**
- No harmonics or overtones (clean single frequency)
- Precise frequency targeting
- Minimal auditory fatigue
- Scientific/premium positioning

**How to verify:**
1. Open in spectrum analyzer (Audacity, Logic Pro)
2. Should show single peak at target frequency
3. No additional peaks (no harmonics)
4. Clean, narrow frequency band

### Tolerance

**Acceptable deviation:** ±1 Hz  
**Example:**
- Target: 963 Hz
- Acceptable: 962-964 Hz
- Reason: Human ear cannot distinguish <3 Hz difference at this range

### Amplitude

**Level:** -12 dB (comfortable, not fatiguing)

**Why -12 dB (not -6 dB)?**
- Quieter than typical music (-6 to 0 dB)
- Non-fatiguing for repeated use (every 4-8 seconds)
- Allows user to set own volume preference
- Mixes well with background music (headroom)
